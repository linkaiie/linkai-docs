LinkAI Route is an OpenAI-compatible gateway with a console for projects, API
keys, credits, usage, and logs. Use it like OpenRouter: one key, one endpoint,
and full visibility into billing and traffic.

## Base URLs

- Console: `https://route.linkai.ie`
- API: `https://api.linkai.ie`

## Key concepts

- **Project:** The unit of ownership and billing.
- **API key:** Authenticates requests; shown once and stored as hash + prefix.
- **Credits:** Balance used to pay for token usage.
- **Rate limits:** RPM, concurrency, and daily caps per project.
- **Logs & usage:** Per-request logs and aggregated usage in the console.

<Callout type="info" title="OpenAI-compatible">
  LinkAI Route accepts OpenAI-style chat completion requests. Provider-specific
  fields are stripped from responses by default to keep compatibility.
</Callout>
